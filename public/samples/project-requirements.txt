Maverick Cloud Migration Project - Certification Requirements

Project Overview:
The Maverick organization is undertaking a large-scale cloud migration initiative to move all on-premise infrastructure to Microsoft Azure. The project spans 18 months and requires a team of 50 engineers with current Azure certifications.

Required Skills and Certifications:

1. Azure Administrator (AZ-104) - Mandatory
   - 20 engineers must hold current AZ-104 certification
   - Skills: Azure AD, Virtual Networks, Storage, Compute, Monitoring
   - Priority: Critical
   - Timeline: Must be certified before Phase 1 (Q1 2026)

2. Azure Solutions Architect (AZ-305) - Required
   - 8 senior engineers must hold AZ-305 certification
   - Skills: Design identity governance, design data platform, design business continuity, design infrastructure
   - Priority: High
   - Timeline: Before Phase 2 (Q2 2026)

3. Azure AI Engineer (AI-900 / AI-102) - Preferred
   - 5 engineers for the AI/ML workload team
   - Skills: Azure Machine Learning, Cognitive Services, Bot Services, Knowledge Mining
   - Priority: Medium
   - Timeline: Before Phase 3 (Q3 2026)

4. DevOps Engineer (AZ-400) - Required
   - 10 engineers for CI/CD pipeline management
   - Skills: Azure DevOps, GitHub Actions, Infrastructure as Code, Container Orchestration (Docker, Kubernetes)
   - Priority: High
   - Timeline: Before Phase 1

5. Data Engineer (DP-203) - Required
   - 7 engineers for data migration workloads
   - Skills: Azure Data Factory, Azure Synapse Analytics, Azure Data Lake Storage, Spark, Python
   - Priority: High
   - Timeline: Before Phase 2

Budget:
- Certification voucher budget: $75,000 USD
- Training materials: $25,000 USD
- Practice exam subscriptions: $10,000 USD

Success Criteria:
- 90% first-attempt pass rate
- All certifications completed within 12 months
- Zero voucher waste (every voucher used productively)
- Candidate readiness score above 70 before voucher issuance

Compliance Requirements:
- All candidates must complete internal training before certification attempt
- Assessment score must be 70% or above before voucher allocation
- Voucher usage must be tracked and reported monthly
- Expired or unused vouchers must be reclaimed within 30 days
